## GEO Writing Rules

- Put the direct answer in the first 80 words.
- Use one strong H1 and clear H2 sections.
- Answer one intent per page.
- Include short FAQ blocks with exact user wording.
- Add concrete facts, not generic marketing copy.
- Mention materials, use cases, limitations, and safety.
- Link from informational pages to the best matching collection or product.

## On-Page Standards

- One canonical URL per topic.
- One primary keyword cluster per page.
- Add `BreadcrumbList` schema to every important landing page.
- Add `FAQPage` schema only where the FAQ is visibly rendered.
- Add `Product` schema to every product page.
- Keep title tags under roughly 60 characters when possible.
- Keep meta descriptions clear, commercial, and human-readable.

## Authority Signals

- Keep contact data identical across site and social profiles.
- Expand About page with founder story and product expertise.
- Add shipping, returns, and materials pages with clear policies.
- Encourage real customer reviews with specific use cases.
- Build mentions from wedding, decor, gifting, and home blogs in Romania.

## Measurement

- Track non-brand clicks in Google Search Console.
- Track indexing coverage and rich result status.
- Track top landing pages by impressions and CTR.
- Track product pages separately from informational guides.
- Review which FAQ and answer blocks generate impressions.
